//// how to execute root command by 'getroot' ////

usageF
push 'getroot' in device, set permission, and execute 'getroot [command]'

> adb push getroot /data/local/tmp
> adb shell
$ cd /data/local/tmp
$ chmod 0755 getroot
$ ./getroot "touch /data/local/tmp/foo.txt"
